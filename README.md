# studymate
스터디 그룹 일정 관리 웹 서비스

## 주요 기능

- 회원가입 및 로그인
- 스터디 그룹 생성 및 관리
- 스터디 그룹 참여 신청
- 그룹장의 참여 승인 및 거절
- 그룹 공용 달력
- 개인 달력
- 일정 등록, 조회, 수정, 삭제
- 그룹장 위임 및 멤버 관리
- 개인 메모
